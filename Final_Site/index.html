Women Programmers

They helped build the web as we know it!

A computer programmer is a skilled professional who codes, tests, debugs, and maintains the comprehensive instructions known as computer programs or code that devices should follow to execute their functions. For those that don't know, code is a language in which computer programs, apps and websites are written. Just as there are many languages spoken around the world, there are also a variety of different programming languages. Historically, women in computing have had an effect on the evolution of the industry, with many of the first programmers during the early 20th century being female. Below are a few to name, but there are many more.


Admiral Grace Hopper


One of the first programmers of the Harvard Mark I computer and invented the first compiler for a computer programming language

She popularized the idea of machine-independent programming languages, which led to the development of COBOL, one of the first high-level programming languages.

She has a Ph.D in Mathematics from Yale in 1934.

Visit here for more information about Admiral Grace Hooper

Margaret Hamilton


Was Director of the Software Engineering Division of the MIT Instrumentation Laboratory, which developed on-board flight software for the Apollo space program.

Founder and CEO of Hamiliton Technologies Inc.

Recieved the Presidential Medal of Freedom from Barack Obama in 2017.

Visit here for more information about Margaret Hamilton

Ada Lovelace


She was the first to recognise that the machine had applications beyond pure calculation, and created the first algorithm intended to be carried out by such a machine

Known as the first computer programmer.

Visit here for more information about Ada Lovelace
